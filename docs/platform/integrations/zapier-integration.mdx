  <p>The <strong>Prompt</strong> action allows you to generate AI-driven responses using your agent.   <ResponseField name="Action Input Properties">

      </ResponseField>

      <ResponseField name="Message" type="string" required>
        The message you want to send to the agent associated with the zapier integration.
      </ResponseField>

      <ResponseField name="Chat ID" type="string">
        The chat ID of the conversation you want to use. If you don't provide one, a new chat will be created.
      </ResponseField>
    </Expandable>
  </ResponseField>

      <ResponseField name="Answer" type="string">
        The response string generated by the agent.
      </ResponseField>
